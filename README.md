# Educação Financeira e Orçamento Pessoal

Este é um um miniguia de estudo sobre finanças e orçamento pessoal feito no NotebookLM

Passos para um Orçamento Pessoal Eficiente (Segundo o Banco Central do Brasil):

Anote Tudo: Registre diariamente todas as receitas e despesas.
Categorize: Agrupe gastos (moradia, alimentação, transporte) para identificar desperdícios.
Analise: Verifique se as despesas estão menores que as receitas ao final do mês.
Planeje: Use os dados para criar uma meta para o próximo mês, priorizando o equilíbrio. 

Pilares da Educação Financeira:
Planejar (PLA): Definir objetivos (curto, médio e longo prazo).
Poupar (POU): Criar uma reserva de emergência (idealmente 3 a 6 meses de custo de vida) para imprevistos.
Crédito Responsável (CRÉ): Evitar juros altos, pagando o total da fatura e parcelando apenas quando necessário.
